# 中文电商售后 Agent 演示问题集

本文件用于前端聊天界面、工具调用链路和课堂展示验收。示例编号来自 `backend/app/data/demo_ecommerce.sqlite`，重新生成数据库后可能变化；可运行 `python scripts/check_demo_db.py` 查看最新样本。

## 工具调用覆盖

| 工具 | 覆盖场景 |
|---|---|
| `query_order_status` | 订单状态、支付、商品明细、物流、手机号后四位校验、售后状态 |
| `query_product_inventory` | 商品名、商品编号、SKU 编号库存查询和补货时间 |
| `evaluate_refund_eligibility` | 待支付、待发货、运输中、已签收、已完成、已有工单等退款资格预判断 |
| `create_after_sales_ticket` | 质量问题、物流破损、少件漏发、未发货退款等工单创建 |
| `query_after_sales_ticket` | 工单状态、流转记录、退款、换货、补偿、附件 |
