# 网络购买商品七日无理由退货暂行办法

## 适用场景

本文件适用于回答以下售后问题：
- 消费者能否申请"七天无理由退货"
- 哪些商品不适用七天无理由退货
- "商品完好"如何界定
- 退货运费由谁承担
- 退货流程和时限要求
- 赠品、积分如何处理

## 核心规则摘要

1. **七天期限**：自消费者签收商品次日起 7 日内可申请无理由退货，无需说明理由。
2. **商品完好**是退货前提，但消费者因检查商品而拆封不影响"完好"认定。
3. **运费承担**：无理由退货的退回运费由消费者承担，经营者另有承诺的除外。
4. **退款时限**：经营者应在收到退货商品之日起 7 日内返还消费者已支付价款。
5. **法定不适用商品**共七类（见下方条款摘录），经营者须在销售时明确标注并取得消费者确认。

## 关键条款摘录

### 不适用七天无理由退货的商品（第六条）

以下七类商品经消费者确认后可不适用七日无理由退货：

1. **消费者定作商品**：按消费者要求定制的商品
2. **鲜活易腐商品**：如生鲜食品、花卉等
3. **数字化商品**：在线下载或消费者拆封的音像制品、计算机软件等数字化商品
4. **报纸期刊**：交付的报纸、期刊
5. **拆封后影响人身健康或安全的商品**：拆封后可能导致商品品质发生改变，影响安全卫生
6. **试用后价值贬损较大的商品**：一经激活或试用后价值贬损较大的商品，如特定电子产品激活后
7. **时效性商品**：销售时已明示临近保质期或有瑕疵的商品

> 说明：第 5、6 类商品需要经营者在销售时经消费者单独确认。未确认的，消费者仍可主张七日无理由退货。

### 商品完好的认定标准（第八条、第九条）

- **商品保持原有品质、功能**，商品本身、配件、商标标识齐全的，视为完好。
- **消费者基于查验需要而打开商品包装**，或者为确认商品品质、功能而进行合理调试，不影响商品完好的认定。
- 以下情况视为商品不完好：
  - 食品、化妆品等必要的一次性密封包装被损坏
  - 服装、鞋帽等商品商标标识被摘除或剪掉
  - 电子电器类商品有未经授权的维修、改动或激活记录
  - 商品受污、受损

### 退货流程与时限（第十条至第十六条）

- 消费者应自签收商品次日起 7 日内向经营者提出退货申请。
- 经营者应当自收到退回商品之日起 **7 日内**返还已支付价款。
- 退款应原路返回，消费者与经营者另有约定的除外。
- 消费者采用积分、代金券等形式支付价款的，经营者应以相应形式返还。
- 退货价款以消费者实际支付金额为准，不含运费。
- 赠品应当一并退回；赠品无法退回的，经营者可要求按标价扣除相应价款。

### 运费承担

- 无理由退货的退回运费由 **消费者** 承担，经营者承诺包退运费的除外。
- 经营者应在商品销售页面明确告知退货费用承担方式。
- 因商品质量问题退货的，运费由 **经营者** 承担。

### 经营者义务

- 经营者应在商品销售页面显著位置标明不适用七日无理由退货的商品。
- 销售不适用无理由退货商品的，应设置显著程序经消费者购买时确认。
- 经营者不得以设置退货流程、自行解释商品完好标准等方式变相限制消费者行使退货权利。

## 对电商售后 Agent 的使用方式

- **七天无理由判断**：先判断商品是否属于七类不适用商品，再判断是否经消费者单独确认
- **商品完好判断**：区分"拆封查验"和"使用后影响二次销售"，前者支持退货
- **退款时效**：告知消费者退款在退货签收后 7 日内到账
- **运费问题**：无理由退货由消费者承担退货运费；质量问题由经营者承担
- **赠品处理**：提醒消费者赠品需一并退回
- **凭证要求**：消费者通常需提供订单号、退货申请时间、商品状态说明

## 注意事项

- 以下情况应引导人工客服处理：
  - 消费者与经营者对"商品完好"存在争议
  - 经营者在销售时未明确标注不适用七天无理由退货
  - 退款超过 7 日未到账
  - 涉及大额商品退款的纠纷
  - 消费者主张的退款金额与实际支付金额不一致
