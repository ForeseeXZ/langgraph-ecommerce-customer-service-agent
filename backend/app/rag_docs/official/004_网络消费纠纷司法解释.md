# 最高人民法院关于审理网络消费纠纷案件适用法律若干问题的规定（一）

## 适用场景

本文件适用于回答以下售后问题：
- 拆封后是否还能七天无理由退货
- 商家"不退不换"声明是否有效
- 二手商品是否适用消费者保护
- 直播间购物遇到售后问题找谁
- 赠品/奖品有问题能否索赔
- 外卖/餐饮外卖平台的售后责任

## 核心规则摘要

1. **拆封不影响退货**：消费者因检查商品拆封且不影响商品完好的，经营者不得以"已拆封"为由拒绝无理由退货。
2. **格式条款无效**：经营者以"签收即视为商品合格""概不退换"等格式条款限制消费者退货权利的，该条款无效。
3. **二手商品经营者**：以营利为目的持续在网络二手市场销售商品的，应认定为经营者，受消法约束。
4. **赠品也担责**：赠品或奖品给消费者造成损害的，经营者应承担赔偿责任，不得以赠品免费为由免责。
5. **直播责任**：直播间运营者应标明实际销售者信息，否则应对商品质量问题承担责任。
6. **外卖平台责任**：网络餐饮服务平台未尽审查义务的，应承担相应责任。

## 关键条款摘录

### 拆封商品与七天无理由退货（第三条）

> 消费者因检查商品的必要对商品进行拆封查验且不影响商品完好的，电子商务经营者不得以商品已拆封为由主张不适用七日无理由退货规定。

- "拆封"不等于"商品不完好"
- 关键是拆封后商品是否仍然完好、能二次销售
- 经营者不能一概以包装已打开拒绝退货

### 格式条款效力（第一条、第二条）

以下格式条款人民法院应认定无效：

1. **收货即视为认可质量**：约定"收货人签收商品即视为认可商品质量符合约定"的
2. **概不退换**：约定"商品一经售出概不退换"的
3. **限制投诉方式**：约定"只能通过特定渠道投诉"变相限制消费者权利的
4. **单方解释权**：约定"经营者享有单方解释权或最终解释权"的

> 说明：这些条款即使消费者勾选同意，法院也不予支持。

### 二手商品经营者认定（第七条）

- 在二手交易平台以营利为目的持续销售商品的，应认定为 **经营者**
- 认定为经营者后，与普通经营者一样适用消费者保护法和三包规定
- 偶尔转让个人闲置物品的个人卖家不属于经营者

### 赠品和奖品的责任（第八条）

> 经营者以赠品或者奖品等形式向消费者提供的商品或者服务，给消费者造成损害的，消费者主张经营者承担赔偿责任的，人民法院应予支持。

- 赠品/奖品同样应符合质量标准
- 赠品造成的损害，经营者应赔偿
- 经营者不得以"赠品免费"为由主张免责

### 直播间购物责任（第十一条至第十三条）

- 直播间运营者应标明实际销售者的名称、地址等信息
- 未标明的，消费者可主张直播间运营者承担销售者责任
- 直播间工作人员的推荐构成虚假宣传的，直播间运营者应承担相应责任
- 网络直播营销平台不能提供直播间运营者真实信息的，承担先行赔付责任

### 网络餐饮服务责任（第十四条、第十五条）

- 网络餐饮服务平台未对入网餐饮经营者进行实名登记、审查许可证的，应承担相应责任
- 入网餐饮经营者不得委托他人加工制作，否则平台和经营者均应承担责任

## 对电商售后 Agent 的使用方式

- **拆封退货纠纷**：消费者反馈商家以"拆封"为由拒退时，引用第三条说明拆封查验不影响退货
- **霸王条款识别**：遇到"概不退换""签收即合格"等条款时，告知消费者该条款无效
- **二手商品**：在二手平台（如闲鱼）购买发现质量问题时，先判断卖家是否为经营者
- **赠品问题**：赠品损坏/有缺陷，告知消费者赠品同样享受售后保障
- **直播购物**：找不到实际卖家时，告知消费者可向直播间运营者主张权利
- **格式条款抗辩**：商家以消费者勾选同意为由拒绝售后时，说明格式条款效力受限
