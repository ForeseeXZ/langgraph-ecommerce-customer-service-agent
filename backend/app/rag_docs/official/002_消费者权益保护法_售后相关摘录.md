# 中华人民共和国消费者权益保护法（售后相关摘录）

## 适用场景

本文件适用于回答以下售后问题：
- 消费者享有哪些基本权利
- 商品质量问题如何处理
- 退货、更换、修理的先后顺序和时限
- 经营者欺诈行为的赔偿标准
- 网络购物中的消费者特别保护
- 格式条款是否有效

## 核心规则摘要

1. **质量保障**：经营者应保证商品或服务符合质量要求，不符合的应承担修理、更换、退货责任。
2. **七日无理由退货**（第二十五条）：网络购物消费者有权自收到商品之日起 7 日内无理由退货，法定除外商品除外。
3. **质量问题的三包责任**（第二十四条）：质量问题 7 日内可退，15 日内可换，超出后经营者仍应承担修理责任。
4. **欺诈赔偿**（第五十五条）：经营者有欺诈行为的，应退还价款并按价款三倍增加赔偿，不足五百元的为五百元。
5. **平台连带责任**：网络交易平台不能提供经营者真实信息的，应承担赔偿责任。

## 关键条款摘录

### 消费者基本权利（第二章）

消费者在购买、使用商品和接受服务时享有以下权利：
- **安全保障权**：享有人身、财产安全不受损害的权利
- **知情权**：享有知悉其购买商品真实情况的权利
- **自主选择权**：享有自主选择商品或服务的权利
- **公平交易权**：享有公平交易的权利，有权拒绝强制交易
- **求偿权**：因购买、使用商品受到损害的，享有依法获得赔偿的权利

### 七日无理由退货（第二十五条）

> 经营者采用网络、电视、电话、邮购等方式销售商品，消费者有权自收到商品之日起七日内退货，且无需说明理由。

- 不适用商品：消费者定作、鲜活易腐、数字化商品、报纸期刊等
- 退回运费由消费者承担
- 经营者应在收到退货后 7 日内返还价款

### 质量问题退货、更换、修理（第二十四条）

> 经营者提供的商品或者服务不符合质量要求的，消费者可以依照国家规定、当事人约定退货，或者要求经营者履行更换、修理等义务。

- **7 日内**：商品不符合质量要求，消费者可要求退货
- **15 日内**：符合换货条件的，经营者应予以更换
- 没有国家规定和当事人约定的，消费者自收到商品之日起 **7 日内**可退货
- **7 日后**符合解除合同条件的，消费者可及时退货
- 不符合解除合同条件的，可要求经营者履行更换、修理义务
- 修理、更换产生的运输等必要费用由 **经营者**承担

### 欺诈三倍赔偿（第五十五条第一款）

> 经营者提供商品或者服务有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，增加赔偿的金额为消费者购买商品的价款或者接受服务的费用的三倍；增加赔偿的金额不足五百元的，为五百元。

- 适用条件：经营者存在欺诈行为（如虚假宣传、以次充好、销售假冒商品）
- 赔偿标准：退还价款 + 价款三倍赔偿（最低 500 元）
- 示例：购买 100 元的商品有欺诈行为，最低赔偿 500 元；购买 5000 元的商品，赔偿 5000 + 15000 = 20000 元

### 网络平台的责任（第四十四条）

> 消费者通过网络交易平台购买商品或者接受服务，其合法权益受到损害的，可以向销售者或者服务者要求赔偿。网络交易平台提供者不能提供销售者或者服务者的真实名称、地址和有效联系方式的，消费者也可以向网络交易平台提供者要求赔偿。

- 平台不能提供经营者真实信息的，承担 **先行赔付** 责任
- 平台明知经营者侵权而不采取必要措施的，承担 **连带责任**

### 格式条款限制（第二十六条）

- 经营者不得以格式条款排除或限制消费者权利
- 不得以格式条款减轻或免除经营者责任
- 不得以格式条款加重消费者责任
- 格式条款有上述内容的，其内容无效

### 个人信息保护（第二十九条）

- 经营者收集、使用消费者个人信息应遵循合法、正当、必要原则
- 应明示收集和使用信息的目的、方式和范围
- 应经消费者同意
- 应公开收集和使用规则
- 不得泄露、出售或非法提供消费者个人信息

## 对电商售后 Agent 的使用方式

- **质量问题判断**：确认商品是否存在质量问题，引用该法判断退货/换货/修理适用
- **欺诈赔偿**：消费者投诉疑似假货时，说明三倍赔偿（最低 500 元）的法律依据
- **平台投诉**：无法联系商家时，告知消费者可向平台主张先行赔付
- **格式条款争议**：消费者提出商家用"概不退换"等格式条款拒绝退货时，说明条款效力
- **个人信息**：消费者对信息处理有疑虑时，引用相关规定
- **期限计算**：质量问题 7 日内可退、15 日内可换的核心期限

## 注意事项

- 以下情况需要人工介入：
  - 消费者主张欺诈三倍赔偿需要举证欺诈行为，不能仅凭客服对话认定
  - 平台先行赔付需要确认经营者确实无法联系
  - 格式条款效力存在争议的情形
  - 涉及人身伤害或大额财产损失
- 本文件不构成法律建议，仅作为自动客服系统的知识参考。
- 实际案件应结合《网络购买商品七日无理由退货暂行办法》等配套法规综合判断。
